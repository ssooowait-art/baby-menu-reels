# 야생의 어둠 — 생존 (Wild Darkness-inspired Survival)

The Wild Darkness에서 영감을 받은 장르 메커니즘의 **오리지널 아이소메트릭 생존 게임**입니다.
모든 그래픽·데이터·코드는 자체 제작(캔버스 절차적 드로잉)이며 외부 에셋이 없습니다.
브라우저만 있으면 실행됩니다 — 별도 빌드/설치 불필요.

## 실행 방법

- `index.html`을 브라우저로 열면 바로 플레이됩니다.
- 또는 로컬 서버: `python3 -m http.server` 후 `http://localhost:8000` 접속.
- 모바일 터치 / 데스크톱 마우스 모두 지원.

## 게임 방법

- **이동/채집**: 땅을 탭하면 이동, 자원을 탭하면 채집합니다.
- **도구**: 나무는 돌도끼 🪓, 바위는 돌곡괭이 ⛏ 가 있어야 캘 수 있습니다.
- **생존 스탯**: HP · 에너지 · 배고픔 · 갈증 · 피로 · 체온을 관리하세요.
  물가를 탭하면 물을 마시고, 음식은 가방에서 먹습니다.
- **낮과 밤**: 밤에는 어둠이 내려오고 **어둠의 정령**이 나타납니다.
  횃불을 들거나 모닥불 곁에 있으면 안전합니다.
- **제작**(⚒): 돌도끼 → 횃불 → 모닥불 → 화덕/침낭 순으로 발전하세요.
  모닥불 근처에서는 요리가 가능하고, 침낭을 탭하면 아침까지 잠을 잡니다.
- **미션**: 좌측 상단의 미션을 따라가면 자연스럽게 생존 기반을 갖추게 됩니다.
- **성장**: 채집·제작·전투로 경험치를 얻어 레벨업(최대 HP/능력치 상승).
- 진행 상황은 브라우저에 **자동 저장**됩니다(설정에서 초기화 가능).

## 구조

```
index.html      # 마크업 + HUD/패널 DOM
css/style.css   # UI 스타일
js/data.js      # 아이템·레시피·미션·몬스터 데이터
js/sprites.js   # 절차적 스프라이트/아이콘 (전부 코드로 그림)
js/game.js      # 월드 생성·이동·채집·전투·낮밤·UI·저장
```
